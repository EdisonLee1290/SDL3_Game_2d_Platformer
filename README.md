# SDL3 2D Platformer (Learning Project)

This project is a small 2D platformer built using SDL3 and C++.  
The main purpose of this project is to begin learning the fundamentals of **game programming**, including:

- Rendering sprites
- Game loops
- Input handling
- Tile-based collision
- Basic animation
- State management
